How the service works:

Write a url on the place below. The service returns a shortened link id to the url you specified.

Alternatively you can make a POST-request to /shorten using Content-Type: application/x-www-form-urlencoded and the parameter name "link".

If you make a GET-request to /{id}, where the id is what was given to you, the service will redirect to you to the original link.
